Info (show/hide)

CONTROLS:

Move: arrow keys
Jump: z key/up arrow
Roll: down arrow key (at speed)
Crouch: down arrow key (static)
Spindash: (while crouching) z key to rev up the spindash, then release down arrow key to perform
Restart: R key
Volume: - and + keys
